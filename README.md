# DeepSkilling

This repository contains training modules and exercises for software development topics.

## What this repository covers

- C#
- Data structures and algorithms
- SQL and databases
- Unit testing
- APIs and backend development
- Angular frontend basics
- Git, DevOps, CI/CD, and Docker
- Generative AI basics

## How to use this repository

1. Start with Module 01.
2. Follow the modules in order.
3. Complete the exercises in each folder.
4. Use the final assessment to review your progress.

- Module 01: Design Patterns
- Module 02: DSA

- Module 08: Angular

- Module 10: Git
- Module 11: DevOps and CI/CD
- Module 12: Docker
- Module 13: GenAI
- Module 14: Final Assessment
