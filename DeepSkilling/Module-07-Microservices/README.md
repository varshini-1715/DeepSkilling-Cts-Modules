# Module 07 - Microservices

This module introduces building applications as small independent services.

## Topics
- Service-based architecture
- Separation of concerns
- Scalability concepts
- Communication between services

## Prerequisites
- Basic API knowledge
- Basic software architecture concepts

## Goal
Understand how large systems can be split into smaller services.
